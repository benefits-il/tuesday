יוצרים את קול-המותג

מתקינים plugin מוכן מהספרייה של קלוד בשם brand-voice, נותנים לו את כל החומרים המפוזרים של Tuesday, והוא מזקק מהם קול אחד עקבי. זו תחנה שכולם עושים.

  1. ב-Cowork נכנסים ל-Customize ואז Personal plugins, מוסיפים את הספרייה anthropics/knowledge-work-plugins ומתקינים מתוכה את brand-voice.
  2. מורידים את חבילת תחנה 1 (למטה) ופותחים אותה. בפנים תיקיית raw עם החומרים המפוזרים של Tuesday: מיילים, פוסטים והכרזות. חלקם טובים, חלקם תאגידיים ומשעממים. זה הבלגן שממנו מזקקים קול.
  3. מצרפים לצ'אט את הקבצים מתוך raw, ומריצים את הפרומפט הזה (הוא מכריח את ה-plugin לעבוד רק מהקבצים שצירפתם):

/brand-voice:discover-brand

משתמשים בקול

כאן מתקינים את ה-plugin הגנרי brand-studio, נותנים לו את brand-voice.md שיצרתם ועוד קבצים מהערכה, ומפיקים תוכן אמיתי. זה המסלול המלא, וכאן פוגשים את כל ארבעת הרכיבים יחד.

מתקינים את ה-plugin (ב-Cowork פותחים /plugin):

/plugin marketplace add benefits-il/brand-studio
/plugin install brand-studio@brand-studio

שני החיבורים — מוסיפים כל אחד פעם אחת ברשימת ה-Connectors. Higgsfield (תמונה ווידאו): Add custom connector, קוראים לו Higgsfield, מדביקים את https://mcp.higgsfield.ai/mcp, Connect, ומתחברים עם חשבון חינם (בלי כרטיס, בלי API key). Gmail (טיוטות): מדליקים את החיבור המובנה. המיילים נשמרים כטיוטה, אף פעם לא נשלחים.

קרא את הקבצים המצורפים האלה לפני הכתיבה:
- brand-voice.md: הנחיות קול-המותג של Tuesday שייצרת בתחנה 1. זה החוק לאיך זה נשמע.

- raw/feature-announcement.md: ההכרזה התאגידית של "Tasks 2.0". קרא אותה רק כדי לשלוף את העובדה האחת האמיתית והשימושית שקבורה בז'רגון. אל תעתיק אף מילה מהטון שלה. הטון הזה הוא בדיוק מה שאתה מציל את המסר הזה ממנו.

נמען: (החליפו בשם, תפקיד, ואיך מצאו אותנו, לדוגמה "מאיה, מעצבת מותג עצמאית, מצאה אותנו דרך ניוזלטר").

כתוב את המייל בעברית כך שהוא:
- נפתח מהצד, לא על המוצר. הקשר, סצנה אמיתית קטנה מטחנת העצמאי, אף פעם לא "רציתי לפנות אליך בנוגע ל-Tuesday".
- מנחית בפשטות את המסר המרכזי האחד מ-outreach-context.md: לקוחות, פרויקטים וחשבוניות במקום אחד, הוא לא יעשה את העבודה, הוא מונע ממך לאבד את החוט, מתחילים חינם בלי כרטיס ובלי שיחת הדגמה.
- נושא את הקול של Tuesday מ-brand-voice.md: יבש, ישיר, מעמית לעמית, קצת מודע-לעצמו. השתמש בהערת-אגב אחת בסוגריים שעושה עבודה אמיתית. בלי באזזוורדס, בלי סימני-קריאה, בלי אימוג'ים, בלי קו-מפריד ארוך.
- נשאר קצר. מתחת ל-130 מילים. שורת נושא שמצדיקה את הפתיחה בלי הייפ.
- נסגר על שורה שמנחיתה וסיבה אחת ברורה ללחוץ, לא "אל תפספס".

כל הטקסט הנראה-לעין במייל חייב להיות בעברית: שורת הנושא והגוף בעברית. שמות המוצר Tuesday ושמות הקבצים נשארים באנגלית.
