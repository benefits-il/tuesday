מה יש בתוך ה-plugin

סקילים

חמישה — אחד לכל מלאכה

השיטה. כל סקיל קורא את דף-המותג שאתם נותנים — צבעים, טיפוגרפיה, לוגו, קול — ומחיל אותו. גנרי: החלפת ערכת-המותג מפעילה אותו על מותג אחר.

חיבורים

שניים, בשתי דרכים

‏Higgsfield לתמונה ווידאו, מחוּוט ב-.mcp.json. ‏Gmail לטיוטות, מדליקים פעם אחת ב-Connectors. הדלת אל מה שבחוץ.

פקודות

חמש

הכפתורים. /outreach /welcome /post /carousel /reel — כל פקודה מפעילה את הסוכן הנכון.

סוכנים

שניים

הדמויות. content-writer כותב את המיילים והכיתובים; visual-producer מפיק את התמונות, הקרוסלה והסרטון.

ה-plugin הוא הקופסה. הוא עוטף את שלושת האחרים. זה כל הרעיון.

התקנה

מתקינים ב-Cowork

ב-Cowork, פותחים את /plugin ומריצים את שתי השורות:

/plugin marketplace add benefits-il/tuesday
/plugin install tuesday-studio@tuesday

שני החיבורים — מוסיפים כל אחד פעם אחת ברשימת ה-Connectors (ב-claude.ai: Settings ואז Connectors · ב-Cowork: Customize ואז Connectors). Higgsfield (תמונה ווידאו): בוחרים Add custom connector, קוראים לו Higgsfield, מדביקים את הכתובת https://mcp.higgsfield.ai/mcp, לוחצים Connect, ומתחברים עם חשבון Higgsfield — חינם, בלי כרטיס, בלי API key. ה-plugin כבר מפנה לכתובת הזו ב-.mcp.json; ההתחברות היא מה שמרשה לו לעבוד. Gmail (טיוטות): מדליקים את החיבור המובנה ומתחברים. המיילים נשמרים כטיוטה, אף פעם לא נשלחים.

What's inside the plugin

Skills

Five — one per job

The method. Each skill reads the brand sheet you give it — colors, type, logo, voice — and applies it. Generic: swap the brand pack and it works for another brand.

Connectors

Two, wired two ways

Higgsfield for image and video, wired in .mcp.json. Gmail for drafts, turned on once in Connectors. The door to the outside.

Commands

Five

The buttons. /outreach /welcome /post /carousel /reel — each one drives the right agent.

Sub-agents

Two

The personas. content-writer writes the emails and captions; visual-producer makes the images, the carousel, and the video.

Plugin is the box. It wraps the other three. That is the whole idea.

Install

Install it in Cowork

In Cowork, open /plugin and run the two lines:

/plugin marketplace add benefits-il/tuesday
/plugin install tuesday-studio@tuesday

Add each connector once in your Connectors list (claude.ai: Settings then Connectors · Cowork: Customize then Connectors). Higgsfield (image and video): choose Add custom connector, name it Higgsfield, paste the URL https://mcp.higgsfield.ai/mcp, click Connect, and sign in with your Higgsfield account — free, no card, no API key. The plugin already points to this URL in .mcp.json; this sign-in is what authorizes it. Gmail (drafts): turn on the built-in connector and sign in. Emails are saved as drafts, never sent.

How to run it

Play with it

  1. Download the brand pack below and unzip it.
  2. Attach brand-voice.md (the voice you made with the brand-voice plugin), brand-sheet.md, and the brief for the job you picked.
  3. Copy the ready prompt from prompts/ and paste it. The command drives the agent, the agent the skill, through the connector.
